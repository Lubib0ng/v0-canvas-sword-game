# ⚔️ 검 로그라이크 3D

## 프로젝트 소개
React Three Fiber 기반의 1인칭 3D 검 로그라이크 게임입니다. 30개의 웨이브를 클리어하며 점점 강해지는 적들과 보스를 물리치는 것이 목표입니다.

## 개발 기간
2025년 1월 (v0 프로토타입)

## 주요 기능

### 무기 시스템
- **롱소드**: 중거리, 균형잡힌 공격속도와 데미지 (쿨다운 500ms)
- **단검**: 근거리, 빠른 공격속도, 낮은 데미지 (쿨다운 250ms)
- **대검**: 광범위, 느린 공격속도, 높은 데미지 (쿨다운 1000ms)
- **전기톱**: 근거리, 상시 자동 공격 (쿨다운 100ms)

### 적 타입
- **기본 적**: 일반적인 이동속도와 공격력
- **빠른 적**: 빠른 이동속도로 플레이어를 추격 (웨이브마다 5% 속도 증가)
- **탱커**: 느린 이동속도지만 높은 체력
- **원거리 적**: 맵 가장자리에서 투사체 발사
- **보스**: 10/20/30 웨이브마다 등장하는 강력한 적

### 보스 패턴
- **10웨이브 보스**: 돌진 패턴 - 플레이어를 향해 빠르게 돌진
- **20웨이브 보스**: 순간이동 패턴 - 랜덤한 위치로 텔레포트
- **30웨이브 보스**: 소환 패턴 - 주변에 적들을 소환

### 특수 시스템
- **대쉬 (Q키)**: 무적 프레임을 가진 회피 기술 (쿨다운 1초)
- **성장 시스템**: 보스 처치 시 능력치 대폭 상승
- **웨이브 시스템**: 30개 웨이브, 10웨이브마다 보스 등장

## 사용 기술
- **Frontend**: Next.js 16, React 19, TypeScript
- **3D Graphics**: React Three Fiber, Three.js, Drei
- **Physics**: Rapier (React Three Rapier)
- **Styling**: Tailwind CSS 4
- **게임 로직**: 컴포넌트 기반 구조
  - Game: 게임 메인 루프 및 상태 관리
  - Player: 1인칭 컨트롤 및 물리 시뮬레이션
  - Enemy: 다양한 적 타입 및 AI
  - Weapon: 무기 시스템 및 공격 판정
  - UI: 오버레이 인터페이스

## 조작 방법
- **WASD**: 캐릭터 이동
- **마우스**: 시점 회전 (포인터 잠금 필요)
- **마우스 클릭**: 검 공격
- **Q**: 대쉬 (무적 상태)
- **ESC**: 포인터 잠금 해제

## 실행 방법
```bash
# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 게임 특징
- 몰입감 있는 1인칭 3D 시점
- 각 무기마다 고유한 공격속도 및 범위
- 전기톱 무기의 상시 자동 공격 시스템
- 대쉬를 활용한 전략적 회피 플레이
- 점진적으로 어려워지는 난이도
- 10웨이브마다 고유한 패턴을 가진 보스 전투
- 보스 처치 시 강력한 능력치 상승 보상
- 30웨이브 클리어 시 승리 엔딩

## 배운 점
React Three Fiber를 통한 3D 게임 개발을 경험했습니다. 물리 엔진(Rapier)을 활용한 캐릭터 컨트롤과 1인칭 카메라 시스템 구현을 익혔고, 3D 공간에서의 충돌 감지와 적 AI 패턴을 구현하며 3D 게임 개발의 기초를 다졌습니다.
