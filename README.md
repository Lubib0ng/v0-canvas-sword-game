# ⚔️ Canvas Sword Roguelike

## 프로젝트 소개
Canvas 기반의 탑다운 검 로그라이크 게임입니다. 30개의 웨이브를 클리어하며 점점 강해지는 적들과 보스를 물리치는 것이 목표입니다.

## 개발 기간
2025년 1월 (v0 프로토타입)

## 주요 기능

### 무기 시스템
- **롱소드**: 중거리, 균형잡힌 공격속도와 데미지 (쿨다운 500ms)
- **단검**: 근거리, 빠른 공격속도, 낮은 데미지 (쿨다운 250ms)
- **대검**: 광범위, 느린 공격속도, 높은 데미지 (쿨다운 1000ms)

### 적 타입
- **기본 적**: 일반적인 이동속도와 공격력
- **빠른 적**: 빠른 이동속도로 플레이어를 추격
- **탱커**: 느린 이동속도지만 높은 체력
- **원거리 적**: 맵 가장자리에서 투사체 발사
- **보스**: 10/20/30 웨이브마다 등장하는 강력한 적

### 보스 패턴
- **10웨이브 보스**: 돌진 패턴 - 플레이어를 향해 빠르게 돌진
- **20웨이브 보스**: 순간이동 패턴 - 랜덤한 위치로 텔레포트
- **30웨이브 보스**: 소환 패턴 - 주변에 적들을 소환

### 게임 진행
- 30개의 웨이브로 구성
- 10웨이브마다 보스 등장
- 보스 처치 시 능력치 대폭 상승
- 30웨이브 클리어 시 승리 엔딩

## 사용 기술
- **Frontend**: HTML5 Canvas, JavaScript (ES6+), CSS3
- **게임 로직**: 클래스 기반 객체지향 구조
  - Game: 게임 메인 루프 및 상태 관리
  - Player: 플레이어 캐릭터 및 조작
  - Enemy: 다양한 적 타입 구현
  - Weapon: 무기 시스템 및 공격 판정
  - UI: 게임 인터페이스 및 정보 표시

## 조작 방법
- **WASD**: 캐릭터 이동
- **마우스 이동**: 공격 방향 조준
- **마우스 클릭**: 검 공격
- **P**: 일시정지
- **R**: 재시작 (게임 오버 시)

## 실행 방법
```bash
# 웹 브라우저에서 index.html 파일 열기
# 또는 로컬 서버 실행
npx serve .
```

## 게임 특징
- 마우스 방향으로 검을 휘두르는 직관적인 전투 시스템
- 각 무기마다 고유한 공격속도 쿨다운
- 점진적으로 어려워지는 난이도
- 10웨이브마다 고유한 패턴을 가진 보스 전투
- 보스 처치 시 강력한 능력치 상승 보상

## 배운 점
Canvas 렌더링 최적화와 게임 루프 설계를 경험했습니다. 클래스 기반 구조를 통해 유지보수가 용이한 코드를 작성하는 방법을 익혔고, 충돌 감지와 적 AI 패턴 구현을 통해 게임 개발의 기초를 다졌습니다.
